# 旅行游记结构模板库

> 根据旅行时长、主题和平台，选择合适的结构模板

---

## 模板一：1-2天短途（单篇流动式）

适用场景：周末小短途、城市一日游、单一景点深度游

```
结构：
开篇场景（200字）
↓
主体（按时间或空间流动，不分章节）
- 关键体验1（200-300字）
- 关键体验2（200-300字）  
- 关键体验3（200-300字）
↓
结尾留白（100字）

推荐总字数：800-1500字
```

**开篇钩子建议**：用当天最精彩的一个瞬间开篇，不必按时间顺序开始

---

## 模板二：3-7天标准行程（按天分章 + 主题串联）

适用场景：国内长途游、东南亚常见行程

```
结构：
标题 + 引言（旅行主题与基调，100-200字）
↓
第一天：[主题] （500-800字）
  - 场景1
  - 场景2
  - 当日感受
↓
第二天：[主题]（500-800字）
...
↓
结尾：[旅程终章] （200-300字）
  - 整体感受
  - 最难忘的一个细节
  - 给读者的话（可选）

推荐总字数：2000-4000字
```

---

## 模板三：7天以上长途（专题分章）

适用场景：深度游、环游、跨国旅行

```
结构：
开篇：旅行的起点与缘由（300字）
↓
第一章：[城市/地区A] （800-1200字）
第二章：[城市/地区B] （800-1200字）
...
↓
美食特辑（500字，可单独成章）
↓
人文印象（500字，可单独成章）
↓
结语：这段旅程教会我的事（200-300字）

推荐总字数：4000-8000字
```

---

## 模板四：主题游（以主题为轴线）

### 4A：美食主题游

```
结构：
开篇：为食而行的宣言（100字）
↓
街头小吃图鉴（3-5种，各100-200字）
↓
一顿难忘的正餐（300-500字详细描写）
↓
当地菜市场探访（200-300字）
↓
美食与文化的连接（200字）
↓
结尾：下次想吃什么

推荐总字数：1500-3000字
```

### 4B：文化/历史探索主题游

```
结构：
开篇：关于历史的震撼（200字）
↓
核心历史遗迹（详细描写，500-800字）
↓
博物馆/纪念馆（300-500字）
↓
与当地人的对话（200-300字）
↓
历史的思考与现代的连接（200字）

推荐总字数：2000-3500字
```

### 4C：自然冒险主题游

```
结构：
开篇：为什么要来这里冒险（150字）
↓
行前准备与期待（100字）
↓
探险过程（按时间顺序，500-800字）
  - 身体感受（疲惫、兴奋、恐惧、成就感）
  - 环境描写（地形、天气、生物）
  - 关键时刻（最艰难的/最美的那一刻）
↓
同行者（如有，100-200字）
↓
事后感悟（100-150字）

推荐总字数：1200-2500字
```

### 4D：摄影旅行记录

```
结构：
追光记：寻找最佳拍摄时机（200字）
↓
镜头里的市井：生活感抓拍（200字）
↓
黄金时刻：日出/日落的等待（300字）
↓
被遗忘的角落：不在游客地图上的地方（200字）
↓
带不走的画面：那些没拍成功但记在心里的（200字）

推荐总字数：1200-2000字
```

## 模板五：平台定制版

### 5A：小红书版游记

**特点**：短段落、多换行、口语化、emoji点缀、实用信息突出

```
结构：
第一段：一句话总结（10-20字，做封面标题）
↓
爆点内容先行（最精彩的部分放最前面）
↓
按逻辑展开（不一定按时间顺序）
↓
实用tips板块（单独列出，用符号标注）
↓
结尾：一句话引发互动的问句

字数：500-1000字
段落：每段3-5行，多空行
标签：#目的地 #旅行风格 #时间 #人群
```

### 5B：公众号版游记

**特点**：有深度、有观点、叙述流畅、适合长读、配图位置清晰

```
结构：
标题（吸引点击，有悬念或有情感共鸣）
↓
引言（交代背景，勾引读者继续读，200字）
↓
正文（分节，每节有小标题，总2000-4000字）
↓
小结（升华主题，100-200字）

格式要求：
- 使用 H2 标题分节
- 每节500-800字
- 穿插[图片说明]提示
- 可设置"推荐阅读"或相关链接
```

### 5C：朋友圈版（超短）

```
结构：
一个场景/感受（2-3句话）
一个实用或有趣的细节（1-2句话）
可选：一个问题引发互动

字数：50-150字
风格：轻松自然，不过度感悟
```

---

## 结构选择决策树

```
用户旅行时长是？
├── 1-2天 → 模板一（单篇流动）
├── 3-7天 → 模板二（按天分章）
│   └── 有主题倾向？ → 可参考模板四相应版本
├── 7天以上 → 模板三（专题分章）
└── 有明确主题 → 直接用模板四

用户发布平台是？
├── 小红书 → 模板五A
├── 公众号 → 模板五B
├── 朋友圈 → 模板五C
└── 未说明 → 标准版（模板一/二/三）
```
