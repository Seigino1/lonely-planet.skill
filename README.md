# ✈️ travel-journal — 旅行游记整理大师

> 碎片化输入 → 一篇有血有肉的旅行游记
> 风格对标 **Lonely Planet**，支持小红书 / 公众号 / 朋友圈多平台输出

---

## 🌟 这个 Skill 是什么

**travel-journal** 是一个为 [WorkBuddy](https://www.codebuddy.cn/docs/workbuddy/Overview) 设计的旅行游记写作助手 Skill。

它读透了 **《Lonely Planet Cambodia》** 的写作风格与叙事逻辑，能够将你旅途中任意形式的碎片记录——一句感叹、流水账、关键词清单，甚至只是"我也说不清楚，就是很感动"——整理成一篇有场景、有情感、有文化深度的专业游记。

---

## 🎯 核心能力

| 输入形式 | 示例 |
|---------|------|
| 一句感叹 | `去了吴哥窟，太震撼了，日出那一刻哭了` |
| 流水账记录 | `第一天金边，第二天暹粒，第三天寺庙，晚上夜市……` |
| 关键词 | `柬埔寨 5天 情侣 吴哥日出 炸蜘蛛 海边` |
| 情感碎片 | `感觉这趟旅行改变了我，说不清楚` |

**输出形式：**
- 📖 完整长篇游记（1000-3000 字）
- 📱 小红书图文版（分段 + emoji + 适合截图）
- 📰 公众号推文版（有标题层级、段落优美）
- 💬 朋友圈版（200 字以内，点睛式）

## 📐 写作原则（对标 Lonely Planet）

Skill 内嵌了 LP 风格的 10 条核心写作原则：

1. **场景先行** — 先带读者进入现场，再表达感受
2. **具体胜过抽象** — "木薯糕外皮微脆，沾椰奶酱甜咸焦糖味" > "当地食物很好吃"
3. **感官锚定** — 视觉、声音、气味、触感，让读者"在场"
4. **历史文化注脚** — 给景点赋予时间厚度，2-4 句，不堆砌
5. **对比与反差** — 古老 vs 现代、沉重历史 vs 街头笑声
6. **数据制造规模感** — "护城河宽达 190 米，绕一圈走近 1 小时"
7. **人物速写** — 向导、摊贩、偶遇的旅行者，2-5 句精准刻画
8. **美食即文化** — 食物是通往地方文化的窗，描述氛围与故事
9. **结尾余味悠长** — 首尾呼应，留下一个具体的画面或问题
10. **保留不确定性** — 留白和疑问，让读者想自己去探索

## 🚀 安装与使用

### 安装方式

将整个 `travel-journal/` 文件夹放入：

```
# 用户级（推荐，全项目通用）
~/.workbuddy/skills/travel-journal/

# 项目级（仅当前项目）
{项目根目录}/.workbuddy/skills/travel-journal/
```

### 触发方式

在 WorkBuddy 对话中，直接说出你的旅行内容，Skill 会自动识别并启动。

**触发关键词举例：**

> 旅行、游记、写游记、整理行程、旅行笔记、旅行总结、旅行分享、
> 碎片化输入整理游记、把我说的整成游记、帮我写旅行故事……

### 使用示例

```
用户：帮我把这次云南旅行整理成游记

用户：去了吴哥窟，太震撼了，日出看了，ta prohm树根很有名，
     还吃了炸蜘蛛，下次还想去

用户：第一天金边，第二天暹粒，第三天寺庙，晚上夜市，
     和男朋友一起去的，柬埔寨5天
```

## 🎨 输出风格调节

游记写完后，你可以继续要求：

- **调整风格**：更文艺 / 更幽默 / 更干货
- **调整篇幅**：精简版（500字）/ 标准版（1500字）/ 详尽版（3000字）
- **适配平台**：小红书版 / 公众号版 / 朋友圈版
- **深化某段**："把吃炸蜘蛛那段写得更有画面感"
